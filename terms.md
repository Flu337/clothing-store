# Техническое задание на разработку сайта для компании «PC-Configurator»

## 1. Введение
Данное техническое задание описывает требования Заказчика по разработке веб-платформы для подбора и реализации кастомных компьютерных сборок. Проект ориентирован на эстетику минимализма и высокую информативность.

## 2. Общие сведения

### 2.1 Назначение системы
Сайт PC-Configurator предназначен для создания профессиональной витрины и экосистемы, где мастера выставляют свои работы, а покупатели находят сбалансированные ПК без торговых наценок ритейла. Основная задача — обеспечить прозрачность сделки и соответствие железа заявленным характеристикам через систему технических аудитов.

### 2.2 Основные конкурентные преимущества компании
* **Монохромная эстетика:** Чистый интерфейс в стиле архитектурных бюро, не отвлекающий от характеристик комплектующих.
* **Технический аудит:** Обязательные отчеты о температурных режимах и стабильности (бенчмарки) для каждой карточки товара.
* **Прямая связь:** Возможность прямого обсуждения конфигурации с автором сборки для индивидуальной настройки.

### 2.3 Целевая аудитория
* **Покупатели:** Геймеры, архитекторы, IT-специалисты (14–45 лет), ценящие надежность и строгий дизайн.
* **Исполнители:** Опытные билдеры и моддинг-студии, нуждающиеся в качественной подаче своего портфолио.

## 3. Структура сайта
На всех страницах в верхней части расположены следующие элементы:
* **Логотип** (черно-белый, ссылка на главную);
* **Меню сайта:** Главная, Каталог, Сравнение, О проекте;
* **Кнопки поделиться** (FB, VK);
* **Кнопка смены языка** (RU/EN).

В нижней части расположены:
* **Копирайт**;
* **Контакты** и ссылки на социальные сети (монохромные иконки).

## 4. Описание экранов

### 4.1 Главная

#### 4.1.1 Прототип
Визуальное решение: крупный типографический заголовок и описание преимуществ слева, интерактивная форма «Задать вопрос» (Ask a Question) справа. В нижней части — три контентных блока: **Mobilise**, **Report**, **Consolidate**.

#### 4.1.2 Описание
Обязательные элементы:
* **Сетка сборок:** Карточки товаров в строгой черно-белой гамме.
* **Форма связи:** Поля «Имя», «Email», «Вопрос» и кнопка отправки (инверсия цвета при наведении).
* **Поведение:** Динамическое изменение контрастности элементов при взаимодействии с курсором.

## 5. Требования к дизайну
* **Стиль:** Строгий минимализм, отсутствие ярких цветов.
* **Цветовое решение:** * **Фон:** Глубокий черный (#000000).
    * **Текст:** Чистый белый (#FFFFFF).
    * **Второстепенные элементы:** Оттенки серого (#888888).
* **Примеры (референсы):**
    * [rog.asus.com](https://rog.asus.com) — визуальный стиль.
    * [pcpartpicker.com](https://pcpartpicker.com) — логика отображения характеристик и карточки товаров.
* **Графика:** Все изображения комплектующих на сайте должны быть выдержаны в единой темной тональности.

## 6. Технические требования
* **Архитектура:** React SPA.
* **Адаптивность:** Mobile-first, корректное отображение на всех типах устройств.
* **Браузеры:**
    * Google Chrome 42+;
    * Safari 9.1+;
    * EDGE 12+;
    * YaBrowser 15+.
* **Разрешения экранов:**
    * Десктоп: 1280 х 768 — 1920 х 1080.
    * Планшет: 1024 х 768.
    * Мобильные: от 360 х 480.
* **Стандарты:** Соблюдение стандартов W3C, оптимизация скорости загрузки (отклик до 200 мс).
